Create is a real estate information platform that makes it easier for development teams to collaborate by bringing research, analysis, and project management together on a single site. Create aggregates streaming data on buildings, properties, market activity, and project economics from leading public and commercial data providers. Create's powerful search tools make discovering new opportunities easy while dynamic analysis helps to drive smarter investment decision making.

Cesium-based Create allows a user to interact within a 3-D environment filled with data, building models, visualizations, and insight regarding real estate development. They have partnered with companies such as CyberCity3D, WalkScore, and DC Open Data to provide the most informative data collection possible.
