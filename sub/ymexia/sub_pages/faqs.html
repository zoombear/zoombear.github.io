FAQs

Q: Where do I access maps of North American ecoregions?

A: For maps depicting the level 1, 2, and 3 ecoregions for North America used as reference for these data, as well as data for use in major GIS applications, please go to the U.S. Environmental Protection Agency North America Ecoregions web site (link will open in new window).

Q: I want to see the EPA ecoregions for Mexico in Google Earth. How do I do this?

A: To view the EPA ecoregions for Mexico in Google Earth, download a KML file of the regions here.

Q: I want to see the Mexico state boundaries in Google Earth. How do I do this?

A: To view the state boundaries for Mexico in Google Earth, download a KML file of the boundaries here.

Q: Why is the State of Puebla identified in the search filter as Pueblo?

A: Available data have been entered as presented in Ynes Mexia's botanical collection journal. When the samples were collected, the State of Puebla was identified as Pueblo.

Q: Some botanical sample points are sitting in the ocean. Why?

A: In order to prevent samples with the same journal-specified location from obscuring each other, the locations for the samples were randomized to sit up to a specified distance away from the sample's defined original location. The distance used to randomize locations varied depending on the sample's identified level of precision. The more precise the location description, the smaller the distance used to randomize the location. This process, when used on samples located on the coast, can generate locations which appear off-shore. The description of the sample location can be found in the sample's information window when the sample point is clicked on.

Q: I used the text field to select samples meeting a sample descriptive I chose, but now want to change the display of the state boundaries. How do I do this?

A: The easiest way to accomplish this is to toggle the state boundaries display and subset the collection using the first (list) search method and then perform any search using the second (text-field) search method.

Q: I am viewing this web site in [...] Internet browser and the display looks unusual. Why?

A: At this time, the Ynes Mexia Botanical Collection web site has been tested for only certain major Internet browsers (Microsoft Internet Explorer, Mozilla Firefox, Google Chrome, and Apple Safari). The display may not behave as expected in other browsers. If you encounter unexpected display or behavior issues, please send a message to YnezMexiaFeedback@gmail.com.

Q: How do I contact the webmaster to report issues?

A: To report issues or concerns regarding this web site, please send an email to the webmaster at YnezMexiaFeedback@gmail.com.
